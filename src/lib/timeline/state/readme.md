## Ответственность

Хранение данных и реактивное состояние

## Правила

- Только `$state` и `$derived`
- Никаких импортов DOM API (window, document, HTMLElement)
- Никаких импортов из других папок timeline (кроме, возможно, logic/constants)
- Классы - простые контейнеры данных с минимальными вычислениями
