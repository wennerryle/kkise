## Ответственность

Рендеринг UI, привязка контроллеров к элементам, передача props.

## Правила

- Получают TimelineContext через getTimelineContext()
- Создают контроллеры, передавая им ctx + идентификаторы
- Минимум логики в \<script\> - только `$derived` для визуальных вычислений (left, width)
- Не вызывают функции из logic/ напрямую
