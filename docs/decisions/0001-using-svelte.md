# Choosing UI framework for simplyfing web-development

## Context and Problem Statement

The UI frameworks allow user to build UI as building blocks through using
components. Without them, we will have to create our UI framework ourselves.

## Considered Options

- Svelte
- Vue
- Angular
- React

## Decision Outcome

Choosed Svelte, because:

- It's declarative by default
- Much simpler than main-stream frameworks like React, Vue, or Angular for the
  beginner

### Consequences

- Relatively small ecosystem to React
